# Step 1
Open game with 512x512 resolution

# Step 2
Fiddle with FOV to make the blocks at the edge of the screen line up

Forwards
```
/tp @p ~ ~ ~ 0 0
```
Directly down
```
/tp @p ~ ~ ~ 0 90
```
